# TeachFi Landing Page

Jekyll landing page for **TeachFi** — a Solana-powered DeFi learning platform for schools.

## Run locally

```bash
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000.

## Build

```bash
bundle exec jekyll build
# Output in _site/
```

## Structure

- `_config.yml` — site config
- `_layouts/default.html` — base layout (nav + footer)
- `index.html` — landing page content
- `assets/css/style.css` — styles (dark theme, Solana gradient)
